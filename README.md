### Clock shop website 
Hello, it's my second big project and its aim is teaching and self-development.

## Getting Started

To run this project locally, you need to install the dependencies in both the backend and frontend directories.

### Prerequisites

* **Node.js** (LTS version recommended)
* **Yarn** package manager

### Installation

1. **Clone and check branch:**
   ```bash
   git clone <your-repository-url>
   cd clock shop
   git checkout main
   ```

2. **Install Root dependencies:**
   ```bash
   yarn install
   ```

3. **Install Backend dependencies:**
   ```bash
   cd backend
   yarn install
   cd ..
   ```

4. **Install Frontend dependencies:**
   ```bash
   cd frontend
   yarn install
   cd ..
   ```

### Running the Application

yarn dev - start app

yarn dev:back - start only backend of app

yarn dev:front - start only front of app

## Others commands

yarn server - start only backend of app without command "tsc -w".

yarn watch - start command "tsc -w" in backend of app.
